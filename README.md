# Naninovel-Sprite-Assembler
NaninovelのスプライトをPython上で組み立てて再現する。
<img width="800" height="687" alt="image" src="https://github.com/user-attachments/assets/1eda8a84-6fab-4420-b99b-9ad8c5a4f836" />  
（なんとなく著作権に配慮してモザイクを入れてます。）

## 使い方
コマンドライン引数は無いです
Assetフォルダを読み込んで開いて表示するスプライト選ぶ

## 動作確認環境
- Python 3.8.10
- Windows 11 Pro / MacOS Ventura
